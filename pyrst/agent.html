agent module¶

Author: Adam White, Matthew Schlegel, Mohammad M. Ajallooeian Purpose: for use of Reinforcement learning course University of Alberta Fall 2017

An abstract class that specifies the Agent API for RL-Glue-py.

class agent.Agent[source]¶

Bases: object

Implements the agent for an RL-Glue environment.

Note

agent_init, agent_start, agent_step, agent_end, agent_cleanup, and agent_message are required methods.
